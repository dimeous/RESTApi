Ниже описаны задачи для Back и Front разработчиков, если вы претендуете на позицию Full Stack, значит вам повезло больше всех,
и вы можете победить сразу обе задачи одним махом :)


1) Реализовать RESTfull API, которое будет содержать следующие возможности:

- получить список резюме
- получить список вакансий
- получить конкретное резюме по ID со всеми связями
- получить конкретную вакансию по ID со всеми связями
- получить подходящие резюме под вакансию (использовать поля опыта работы и мест работы)
- получить подходящие вакансии под резюме (использовать поля опыта работы и мест работы)

Можно использовать чистый php или любой фреймворк. В качестве источника данных необходимо использовать файлы resumes.json,
 vacancies.json, works.json,
  источник данных может меняться. Авторизация не нужна.

2) Реализовать фронтенд часть для приложения. Желательно использование reactjs или angular 2+.
Дизайн произвольный. Доступ к файлам-источникам реализовать через http запрос (если Вы не писали апи).

- страница со списком резюме и фильтрами по всем возможным полям
- страница со списком вакансий и фильтрами по всем возможным полям
- страница с конкретным резюме и подходящими вакансиями
- страница с конкретной вакансией и подходящими резюме
- страница с фильтрами для подбора вакансий
- страница с фильтрами для подбора резюме
